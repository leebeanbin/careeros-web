# CareerOS 레이아웃 시스템

CareerOS는 dark neutral 기반의 어시스턴트 작업 공간입니다. 장식적인 SaaS 랜딩보다 조용하고 읽기 쉬운 운영형 화면을 우선합니다.

## 앱 셸

```
┌──────────────┬────────────────────────────────────┐
│ Sidebar      │ Topbar                             │
│ 230px        ├────────────────────────────────────┤
│ dark rail    │ Page content                       │
│              │ workflow별 max-width               │
└──────────────┴────────────────────────────────────┘
```

- 루트 앱 배경: `rgb(8,9,10)`.
- 사이드바: `rgb(11,12,13)`.
- 페이지 표면: `rgb(13,14,15)` 또는 `rgb(17,18,19)`.
- 경계선: `rgba(255,255,255,0.06)`.
- 카드/패널 radius는 `8px` 이하를 기본으로 합니다.

## 페이지 리듬

- 복잡한 워크플로우 페이지는 상단에 `AgentIntro`를 둡니다.
- 상태 요약이 실제 판단에 도움이 될 때만 status strip을 사용합니다.
- 목록은 큰 카드보다 밀도 있는 row를 우선합니다.
- sticky header는 `48px` 높이의 compact control bar로 유지합니다.

## 반응형 규칙

- 좁은 workflow는 `max-width`와 중앙 gutter를 사용합니다.
- 데이터 구조상 필요한 경우에만 가로 스크롤을 허용합니다.
- row 안의 텍스트는 버튼을 밀어내지 않고 truncate 처리합니다.

## 인증 페이지

로그인/회원가입은 최소한의 public entry로 유지합니다. 앱과 같은 타이포그래피와 muted link 처리를 사용하고, 과한 브랜드 색 표면은 피합니다.
